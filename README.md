# AI Bubble Tracker

An English-first neutral research dashboard for tracking AI bubble signals across GPU, CPU, memory, and storage stocks.

## Local Development

```bash
npm install
npm run dev
```

## Cloudflare Pages

- Framework preset: Astro
- Build command: `npm run build`
- Build output directory: `dist`
- Node.js version: 20+

The first version uses monthly manual data updates. Update the files in `src/data/` and rebuild.

Production domain: `https://aibubbletracker.top`
